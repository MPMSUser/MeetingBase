Funktion zum Kopieren von Mandanten (bzw. Umbauen von Mandanten-Tables auf eine andere Mandantennummer)

  • Schritt 1: (Soweit noch nicht erfolgt): Neuanlegen des Ziel-Mandanten, Emittenten-Daten mit betterMeeting pflegen.
  • Schritt 2: Im Quell-Mandant Status auf "Komplett Teststatus" setzen.
  • Schritt 3: Schema des Quell-Mandanten aus dessen Datenbankschema mit MySQL Workbench exportieren (Achtung: 'Include Create Schema' NICHT aktivieren!)
  • Schritt 3: Den in Schritt 3 erzeugten Export in Schema des Ziel-Mandanten mit MySQL Workbench importieren (Achtung: Ziel Schema in 'Default Target Schema' auswählen)
  • Schritt 4: Ggf. Formular-Designs des Mandanten kopieren (Dateiebene, betterMeeting_Root\betterreports\MxxxJxxxxxx)
  • Schritt 5: Diese Funktion - Mandantennummer ändern - im Ziel-Mandant durchführen
  • Hinweis: einige wenige (eher unbedeutende) Tables werden bei dieser Methode nicht kopiert, z.B. Fehlertexte für Portal
